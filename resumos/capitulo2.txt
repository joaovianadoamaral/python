os dados de mensagem tem delimitadores.
as aspas são os delimitadores de mensagem tanto aspas simples quanto aspas duplas
existe uma preferencia na comunidade de aspas simples

ex:  'mensagem'.

todos os comandos em python são funções:

ex de função : print()
éla indica para mostrar uma mensagem na tela 
print('mensagem')

podemos mostrar números na tela e números não tem aspas

exemplo de calculo usando print(7+4)

caso usarmos print('7'+'4'), o resultado é 74 
pois concatenamos duas mensagens.
podemos usar uma virgula também.

para concatenar use: + ou ,

o mais só serve para duas mensagens, caso ao contrário usaremos ','
